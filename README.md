# 🎥 猫眼票房插件

实时查询猫眼电影票房排行榜的AstrBot插件

## 功能特性
- 实时获取TOP5票房数据
- 自动转换票房单位（万→亿）
- 显示排片率/上座率等核心指标
- 错误处理与友好提示

## 使用方式
发送指令：
/票房排行
